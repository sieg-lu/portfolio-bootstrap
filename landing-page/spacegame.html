Space Crap Game

Battle the enemies in random generated maze.
Shoot like FPS but command your allies like RTS and TowerDefense!





Map Generation


As the picture above, Room 1, 2, and 3 are hollowed from the solid space. It's easy to implement this: when create a new random room, check if it's overlapping with other rooms using AABB collision detect.

Next step, we choose two rooms to connect. Here, Room 1 and 2's centers can be connected by Path A and B, simply choose A or B randomly.

Repeat hollowing the rooms and connecting the rooms by specified times.

PathNodes Generation


The PathNodes are needed for the actors to go to the destinations.

First, we place the PathNode on the KEY points (rooms' centers & corridors' corners).

Then, shoot a ray from each point to other points, see if the ray is intersecting with the walls (remember the raytracing, huh?), if not, we can put these two PathNodes into the Connected Graph.

As shown above, from PathNode A, the green rays mean connected, while the red rays are intersecting with the walls.

Path Finding

Here we used the standard A* searching algorithm (Breadth First Search USING Priority Queue WITH Heuristic Function). Heuristic Function is the distance to the target PathNode.
